# 🎯 [프로젝트 명: GraphoVision] 필기 기반 심리분석 서비스 기획(PRD) 및 UI 디자인 문서 작성 요청

너는 지금부터 서비스 기획자(PO)이자 수석 UI/UX 디자이너 역할을 수행할 거야. 
현재 필기 이미지로부터 8가지 심리 지표를 도출하는 Custom CNN 딥러닝 모델의 PoC(개념 증명)를 완벽하게 성공한 상태야. (검증 정확도 74.9% 확보, 8각 레이더 차트 시각화 연동 완료)

이제 이 AI 엔진을 탑재하여 사용자들이 웹에서 재미있게 즐길 수 있는 '필기 심리검사 B2C 웹 서비스'를 론칭하려고 해. 아래 요구사항과 페이지 스펙을 바탕으로, 개발자와 디자이너가 바로 작업에 착수할 수 있도록 [1] 서비스 기획 문서(PRD)와 [2] UI/UX 디자인 상세 스펙 문서를 마크다운 형태로 통합하여 작성해 줘.

---

## 🚀 1. 서비스 핵심 콘셉트 및 비즈니스 모델 (BM)
1. 서비스 본질: 사용자가 직접 쓴 글씨 사진을 업로드하면, AI가 필체를 분석하여 8가지 성격 지표를 도출하고, LLM이 맞춤형 심리 멘트를 생성해 주는 엔터테인먼트 심리검사 웹 플랫폼.
2. 유료화 모델 (재화 시스템): 
   - 검사 1회 수행 시 '1 크레딧(재화)'이 차감됨.
   - 1 크레딧의 가치는 소비자 가격 '1,000원'으로 산정함.
   - 이에 따른 재화 결제 및 충전 시스템이 필수적으로 존재해야 함.

---

## 💻 2. 필수 구현 페이지 및 요구 스펙 (UI/UX 정보 구조도 포함)

너가 작성할 문서에는 아래 6개 핵심 페이지의 '기획 스펙'과 '화면 UI 컴포넌트 구성 요소', '사용자 시나리오'가 상세히 포함되어야 해.

### ① 자체 로그인 및 회원가입 페이지 (Sign In / Up)
- 간편한 자체 회원가입 (소셜 로그인 연동 안 함, 이메일/비밀번호/닉네임 기반의 독립 시스템).
- 로그인 성공 시 메인 화면으로 리다이렉트 및 보유 재화(크레딧) 세션 연동.

### ② 메인 및 AI 심리검사 페이지 (Main / Test)
- 사용자가 카메라로 직접 사진을 찍거나 앨범에서 필기 사진을 업로드하는 영역.
- 이미지 전처리 및 AI 분석 중임을 알려주는 모던한 로딩 애니메이션.

### ③ 검사 결과 및 시각화 페이지 (Result)
- AI 모델이 도출한 8가지 지표를 보여주는 **'8각 레이더 차트(Radar Chart)'** 렌더링 영역.
- AI 스코어를 기반으로 사용자 맞춤형 분석 문장을 위트 있고 자연스럽게 풀어주는 **'LLM 생성 멘트(리포트)'** 영역.
- 결과를 카카오톡이나 SNS로 공유할 수 있는 공유 버튼.

### ④ 친구와의 필기 궁합 페이지 (Compatibility / Chemistry)
- 두 사람의 필기 분석 결과를 매칭하여 성격적 조화도, 시너지 점수 등을 보여주는 페이지.
- 나와 친구의 레이더 차트를 오버레이(겹쳐서) 비교해 주는 시각화 UI 포함.
- (1인 검사 완료 데이터 기반 또는 2인 동시 업로드 방식 제안 필요).

### ⑤ 재화 충전 페이지 (Billing / Credit Shop)
- 1 크레딧 = 1,000원 기준의 결제 패키지 라인업 (예: 1회권 1,000원 / 5회권 4,500원 등 할인율 적용 제안).
- 토스페이먼츠, 카카오페이 등 간편결제 연동을 고려한 충전하기 UI 버튼 및 결제 수단 선택 영역.

### ⑥ 마이페이지 및 과거 기록 조회 (My Page / History)
- 현재 사용자의 프로필(닉네임) 및 보유 재화(크레딧) 잔액 표시.
- **과거 검사 히스토리 리스트:** 내가 이전에 검사했던 기록들을 리스트로 보여주고, 클릭 시 당시의 결과 페이지(레이더 차트 + LLM 멘트)를 다시 볼 수 있는 기능.

## 🎨 3. UI/UX 디자인 가이드라인 요구사항
- **톤 앤 매너:** 사용자가 '심리 검사'라는 전문성을 느끼면서도 '재미' 요소로 가볍게 즐길 수 있도록, 너무 무겁지 않고 세련된 뉴트럴 친화적 톤(예: 차분한 크림/베이지 바탕에 포인트 컬러 사용 등) 제안.
- **컴포넌트 중심 설계:** React와 Tailwind CSS로 쉽게 퍼블리싱할 수 있도록 각 페이지별 UI 요소를 블록 단위(Card, Button, Modal 등)로 명확하게 쪼개서 설명해 줄 것.
- **차트 렌더링 스펙:** 웹 환경에서 레이더 차트를 끊김 없이 렌더링하기 위한 프론트엔드 라이브러리(Chart.js 또는 Recharts 등) 추천 및 연동 방식 가이드 포함.

위 내용들을 종합해서 아주 방대하고 디테일한 기획 및 디자인 문서를 뽑아줘!
